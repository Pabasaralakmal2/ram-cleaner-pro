# 🚀 RAM Cleaner Pro

![Version](https://img.shields.io/badge/version-2.0-blue)
![Platform](https://img.shields.io/badge/platform-Windows-lightgrey)
![License](https://img.shields.io/badge/license-MIT-green)

A powerful, stylish command-line RAM cleaner with real-time memory monitoring and multiple cleaning levels.

## ✨ Features

- **Real-time RAM Monitor** - Live memory usage display with visual progress bar
- **4 Cleaning Levels**:
  - 🧹 **Quick Clean** - Empty working sets and flush DNS
  - 🔧 **Normal Clean** - Clear system cache and temp files
  - ⚡ **Deep Clean** - Aggressive memory optimization
  - 🚀 **Ultra Clean** - Maximum system performance
- **Stylish Console Interface** - Box-drawing characters and colors
- **Memory Statistics** - Track how much RAM you've freed
- **One-Click Operation** - Simple number key interface

## 🛠️ Installation

### Method 1: Quick Install
1. Download `RAMCleaner.bat` and `install.bat`
2. Right-click `install.bat` and select "Run as administrator"
3. Follow the installation wizard

### Method 2: Manual Run
1. Download `RAMCleaner.bat`
2. Double-click to run directly (no installation needed)

## 🎮 Usage

1. Launch RAM Cleaner Pro
2. Monitor your current RAM usage
3. Select cleaning level by pressing:
   - `1` - Quick Clean
   - `2` - Normal Clean
   - `3` - Deep Clean
   - `4` - Ultra Clean
   - `5` - View Statistics
   - `6` - Settings
   - `7` - Exit

## ⚙️ Cleaning Levels Explained

| Level | Actions | Duration | Impact |
|-------|---------|----------|---------|
| **Quick** | DNS flush, empty working sets | ~2 seconds | Low |
| **Normal** | + Clear prefetch, temp files | ~5 seconds | Medium |
| **Deep** | + Clear browser caches, thumbnails | ~30 seconds | High |
| **Ultra** | + Memory optimization, disk cleanup | ~2 minutes | Maximum |

## 🔧 Requirements

- Windows 7/8/10/11
- Administrator privileges (for Deep/Ultra clean)
- No additional software needed

## 📊 Statistics Tracking

The program tracks:
- Number of cleanups performed
- Total memory freed
- Per-level usage statistics
- Current RAM usage

## ⚠️ Safety Notes

- Quick and Normal cleans are completely safe
- Deep and Ultra cleans may close some applications
- Always save your work before Deep/Ultra cleaning
- The program won't delete essential system files
